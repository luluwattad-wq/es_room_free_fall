اكتبوا معادلتَي الموقع:

⚡ سؤال التحقق

زمن اللقاء = 3 s
3 s < 3.46 s → إذاً...؟
🏆 سؤال الكود النهائي

احسب مجموع سرعتَي الجسمين عند الالتقاء مع مراعاة الإشارة الكاملة:
(vA) + (vB) = ؟

💡 الكود = القيمة المطلقة لهذا المجموع (بدون إشارة)
